NEW API ENDPOINT needed in FastAPI (add to api/routes.py):

GET /api/v1/ragas/latest
  Query ragas_history table, return most recent run:
  {
    evaluated_at: string,
    faithfulness: float,
    answer_relevancy: float,
    context_precision: float,
    context_recall: float,
    total_queries: int,
    passed_queries: int,
    alert_fired: bool,
    run_label: string
  }

GET /api/v1/ragas/history
  Query ragas_history last 30 runs, return array:
  [{ evaluated_at, faithfulness, answer_relevancy,
     context_precision, context_recall, alert_fired }]

GET /api/v1/ragas/run
  Trigger a fresh RAGAS evaluation (runs run_nightly_eval.py).
  Returns: { status: "running" | "completed", message: str }
  Run as background task — do not block the response.
  Use FastAPI BackgroundTasks for this.

THRESHOLD CONSTANTS (define at top of file):
  const THRESHOLDS = {
    faithfulness:      0.80,
    answer_relevancy:  0.85,
    context_precision: 0.75,
    context_recall:    0.80,
  }

  function getStatus(score, threshold) {
    if (score >= threshold)           return 'pass'
    if (score >= threshold - 0.05)    return 'warn'
    return 'fail'
  }

  function getColor(status) {
    return { pass:'#10B981', warn:'#F59E0B', fail:'#EF4444' }[status]
  }
